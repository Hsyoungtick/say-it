### 人工交互验证步骤（需在 `npm run tauri:dev` 中执行）

1. 录音转写：点击上传区选择文件；从资源管理器拖放音视频到窗口，确认拖入高亮、松开后加载文件信息；选中后点「重新选择」；未选文件时「开始识别」禁用，选中后可点击；无 API Key 时出现「去设置 API Key」；识别中出现「取消」并可中断；`StatusBar` 圆点随 idle/进行/完成/错误变化。
2. 结果区：切换「纯文本/字幕」；点「复制文本」；有句级时间戳时「导出 SRT」可用并能保存文件。
3. 文稿对齐：选择/拖放音频；粘贴多行文稿，确认「N 行有效文稿」计数；「开始对齐」在缺文件或空文稿时禁用；对齐完成后切换「完全按文稿 / 识别修正」，确认低匹配行黄色标注、识别补入行蓝色标注，导出 SRT 正常。
